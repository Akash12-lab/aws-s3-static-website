# 🚀 AWS S3 Static Website Hosting

## 📌 Project Overview

This project demonstrates how to host a static website using Amazon S3.
The website is deployed using AWS Free Tier and is publicly accessible via an S3 endpoint.

---

## 🛠️ Technologies Used

* Amazon S3
* HTML
* CSS

---

## 📂 Project Structure

```
aws-s3-website/
│── index.html
│── style.css
│── README.md
```

---

## 🚀 Steps Performed

1. Created an S3 bucket with a unique name
2. Uploaded website files (index.html, style.css)
3. Configured bucket policy to allow public access
4. Enabled static website hosting
5. Accessed the website using S3 endpoint
